## Key Lessons Learned

1. **Static files embedded via include_str!() cost 3 min per JS edit.** Add OPENFANG_STATIC_DIR before iterating on client code.
2. **Two implementations of the same thing will diverge.** voice.html and chat.js had different WS send paths, different state tracking, different bugs. Consolidate first.
3. **Verify safetensors weight names against the actual file.** Don't assume architecture from documentation — inspect with `safe_open().keys()`.
4. **Server-side VAD during TTS playback is unreliable.** Ambient noise triggers false positives with both neural and energy detection. Client-side energy VAD is the correct approach.
5. **Prefer uv over pip** on all cluster nodes.
