## Key decisions

| Decision | Rationale |
|----------|-----------|
| `SCHEMALESS` for all tables | Existing data was written schemaless; SCHEMAFULL would silently drop undeclared fields |
| `FLEXIBLE TYPE any` for enum fields | Rust enums serialize as strings (unit variants) or objects (tuple variants). `FLEXIBLE TYPE object` rejected string values. |
| `TYPE RELATION` for `relations` | Already uses RELATE syntax with `in`/`out`; existing records have these fields populated |
| Dropped HNSW KNN query rewrite | 0 embeddings in live DB — rewriting recall is premature |
| Single `.query(SCHEMA_DDL)` call | Atomic, fewer round-trips, easier error handling than 50+ individual statements |
| Release binary copy (not symlink) | Symlink to `target/debug/` meant any `cargo build` silently changed the daemon. Copied release binary is immutable and intentional. |
