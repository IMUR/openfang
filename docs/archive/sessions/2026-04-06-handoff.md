## Decisions Made

- **PCM16 over Opus** — zero client dependencies, works in all browsers. Opus upgrade via `SessionInit` codec field if needed later.
- **Same WebSocket for text and voice** — server already multiplexes binary/text on agent WS connection.
- **GTX 970 abandoned for voice** — CTranslate2 dropped SM 5.2, Maxwell can't do fp16/int8 compute. Not worth building from source.
- **RTX 2080 on drtr for voice** — SM 7.5, Tensor Cores, 8GB, all packages work. Config already supports remote endpoints.
- **Keep CTranslate2 for Whisper** — fastest Whisper engine per research. whisper.cpp and ONNX Runtime are slower.
- **Future: Rust-native voice** — `ort` crate (ONNX Runtime) + `whisper-rs` (whisper.cpp bindings) to eliminate Python services. Research agent identified this as the medium-term path.
